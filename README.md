## super useful command line utility

[![Greenkeeper badge](https://badges.greenkeeper.io/rickycodes/lolify.svg)](https://greenkeeper.io/)

### install:
`npm i -g lolify`

### usage:
```
$ lolify 'hello how are you?'
hallo how era you?
```

also accepts stdin:
```
$ echo 'hello how are you?' | lolify
hallu huw era yui?
```
